# MyWebApp Server

# To run the project in development server

### Install the packages
- Express
- Mongoose 
- jsonwebtoken 
- body-parser 
- dotenv 
- bcrypt

### To get JWT Secret Key
```
> node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"
``

```
> npm install
> nodemon src/app.js
```

# Pre-requites

1. Node JS v12.16.1 or above.
2. Visual Studio Code
3. Postman
